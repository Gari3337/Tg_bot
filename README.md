# Tg_bot
### Бот для записи учеников на репетиторство
Ученик будет записываться в свободный день и время через Telegram, поля в записи: имя, фамилия, дата и время, ученик может отменять записи. Репетитор будет видеть на какое время записались ученики, а также ему будут приходдить сообщение о новой записи, также у репетитора есть возмонжность отменять записи учеников.

Токен преподавателя задается из коружения.
